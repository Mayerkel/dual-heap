# CSCI 331- Dual Heap Project

Dual Heap project

# Building

```
make build
```

# Example Usage

```
./bin/dualheap [args]
```

# Running the tests

## Dependencies

- pkg-config
- gtest

```
./bin/test_dualheap
```
